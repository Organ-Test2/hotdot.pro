Tech

  • Open Source

    Project's source code is available on GitHub under the MIT license

  • three.js

    JavaScript Library which makes WebGL simpler.

  • headtrackr.js

    JavaScript library which uses web camera to track position of user's head.

  • getusermedia();

    JavaScript API, providing access to user's web camera or microphone.

  • WebSocket

    Connection protocol made for real-time message exchange between browser and server.

  • deviceOrientation

    JavaScript API, providing access to device's spatial orientation sensors.

  • Dogecoin

    Cryptocurrency based on Litecoin, which in turn has been forked from Bitcoin.

  • socket.io

    JavaScript library for web apps and real-time data exchange which uses WebSockets or imitates them if they're unavailable.

  • WebGL

    A JavaScript API for rendering interactive 3D and 2D graphics without the use of plug-ins.

  • WebAudio

    JavaScript API which gives full control over sound.
