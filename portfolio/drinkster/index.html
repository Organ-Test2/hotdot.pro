Drinkster Layout and Programming

Mission

We were commissioned to create a website on gastronomy and alcohol consumption culture that would present information in a most user-friendly way – depending on the content format.

We created original algorithm that allows to cut a paragraph in several places randomly to insert information blocks. There’s a great work going on behind the scenes for several blocks to interact seamlessly on one page.

This great idea was implemented with equal excellence. The website design developed by the identity studio Sputnik contains lots of distinguishing features and recognizable details to create the website’s extraordinary style.

Unconventional design solutions together with certain browser and platform limitations demanded delicate technical treatment and meticulous work.

Articles have no template and can be composed of various elements – the way content managers like. Any part of the text may contain unfolding information blocks, glossary, added marker highlighting, non-standard fonts, image galleries. There’re also special article templates: comparison of several drinks and a slide-format article.

The website structure demanded elaborate architecture even for JS client code.

We did our best for the corporate fonts to be displayed in as high quality as possible.

The admin panel was tailored for the project to ensure fast and comfortable work for content managers with lots of useful functions.
